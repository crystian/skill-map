# sm-cli (alias — bin name confusion)

This is **not** a real package. `sm` is the binary name, not the package name. This placeholder is published so that someone searching for "sm-cli" lands on the redirect rather than a third-party squatter.

The official command-line tool is **`@skill-map/cli`**.

## Install the real one

```bash
npm install --global @skill-map/cli
```

That installs the `sm` binary (and its long-form alias `skill-map`) in your `$PATH`.
